# Netflix Dashboard using Tableau

## Overview

This project is an interactive Netflix Dashboard created in Tableau by analyzing Netflix content data. The dashboard provides insights into movies and TV shows available on Netflix, including ratings, genres, release years, and country-wise distribution.

The project was built as part of a Tableau data visualization practice project to improve dashboard development and storytelling skills.

## Dashboard Features

- Total Movies and TV Shows
- Content Distribution by Type
- Ratings Analysis
- Genre Distribution
- Country-wise Content Availability
- Release Year Trends
- Interactive Filters for Better Exploration

## Tools Used

- Tableau
- CSV Dataset
- Data Visualization
- Dashboard Design

## Dataset

The dataset contains information about Netflix titles, including:

- Show ID
- Title
- Type (Movie/TV Show)
- Director
- Cast
- Country
- Release Year
- Rating
- Duration
- Genre

## Key Insights

- Movies represent a larger portion of Netflix content than TV Shows.
- Netflix content has grown significantly over recent years.
- Certain countries contribute more content to Netflix than others.
- TV-MA and TV-14 are among the most common content ratings.
- Drama and International content are among the most popular categories.
